# Roll20
Roll20 Resources

The scripts in this repository are for Roll20 OGL 2014. They require my Farsidelib and both of ScriptCards Creator's (Kurt) 5th Edition and System Nuetral libraries (5Elib and snlib) to work correctly.

Kurt's Scripts and Libraries can be found here: https://github.com/kjaegers/ScriptCards/

I have tried to give credit to all scripts that are not my direct creation. If you find one that you wrote and are not credited, please contact me and I will give credit
or delete if requested.


All saving throw spells look for advantages for a Character and add those into the dice roll.  Ex: Elves have advantage on saving throws for Charm Person. To account for this - in the Global Save Area, create a new Global Save named Charm and set the roll to 0. Leave it unchecked so that it does not appear in any saving throws. The Charm Person spell will check for it and automatically move the save from Disadvantage to Normal or from Normal to Advantage. Barbarians have advantage on Dexterity saves. Follow the same process, only make the name DEX (all caps for abilities). Finally, the same can be done for damage types (necrotic, psychic, etc.)
