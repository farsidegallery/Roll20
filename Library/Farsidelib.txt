--/|ScriptCards Library: Farsidelib
--/|
--/|   far_CustomCritCheck|ReturnStringName
--/|     Are you using a custom critical damage game? 0 = no and 1 = yes
--/| 
--/|   far_PCConcenCheck|PCRollReturnString;BardScriptReturnString
--/|     Used with Concentration Trigger.
--/|     Do you want your players to hit a button for the Concentration Check? (0 = fully automated and 1 = Player will be prompted with a button)
--/|     Are you using FarsideGallery's Bardic Inspiration script? (0 = no and 1 = yes)
--/|
--/|   far_GetAndCheckSlotInformation|tokenid;SpellLevel
--/|     Accepts token-id and spell slot level to verify the spell caster has the desired spell slot available.
--/|
--/|   far_NoSlotsLeft|SpellLevel
--/|      Accepts spell slot level, outputs message to chat explaining there are no spell slots available, and terminates the script.
--/|      
--/|   far_DeductSpellSlot|tokenid;SpellLevel   
--/|      Accepts token-id and spell slot level to deduct a "used" slot from the available slots. Displays output message
--/|      with number of available spell slots remaining.
--/|
--/|   far_GetAvailableSpellLevels|CharacterID;MinimumSpellLevel;ArrayNameToSet
--/|      Builds array of spell level slots avaialble for casting of spell. Accounts for Warlock spell slot as well.
--/|     
--/|   far_FindFeat|tokenid;FeatName;FeatFlag   
--/|      Accepts a token-id to search for the desired FeatName and returns a 1 if present or a 0 if not in the FeatFlag.
--/| 
--/|   far_BuildTargetList|NumberOfTargets;ReturnStringName
--/|      Accepts a number variable defining how many targets will be added to the return target string.
--/| 
--/|   far_AddSeparator|
--/|      Subfunction called by far_BuildTargetList to add delimiter in target string.
--/|
--/|   far_BuildTargetArray|ArrayName;Number of Loops;String Containing target token-ids
--/|      Given a target string (ideally from far_BuildTargetList) creates and populates an array of targets using the  
--/|      supplied array name. Loop building the array will cycle to the number provided. Array will consist of unique 
--/|      (no repeats) target token-ids.
--/|
--/|   far_CheckDamageModifiers|characterid;damageType;damageVariableName;isMagical(0 or 1);IsSilver (0 or 1);IsAdamantine (0 or 1)
--/|      Checks npc (characterid) and looks for resistant, vulnerable, or immue on (damagetype) damage and returns the result in
--/|      (damageVariableName), which is a string that can be added to the end of a roll assignment to modify the damage as appropriate   
--/|
--/|   far_Check_GLOBAL_MOD_EXISTS|characterid; Category; name; returnstring (0=does not exist, 1=exists)
--/|      Checks to see if the specified Global Mod exists and returns string variable
--/|
--/|   far_Check_GLOBAL_ATTRIB_SAVE_EXISTS|characterid; attributename; returnstring (0=does not exist, 1=exists)
--/|      Checks to see if a Global Save Mod exists in any variation of the attribute provided and returns numeric flag
--/|   
--/|   far_PlayEffects|Parameters are : sourcetoken; targettoken; source effect; target effect; line effect; sound effect
--/|      Used to display vfx in one referencing call
--/| 
--/|   far_HexHeal|characterid; token-id
--/|      If creature dies while under effects of HexBlade's Curse AND to the Warlock who cast the curse, heals the Warlock
--/|  
--/|   far_Inspiration|token-id;tokenmarker;attackstring;attackstringreturn
--/|      Used if PC character has Inspiration to alter base roll.
--/|   
--/|   far_InspirationRoll
--/|     Subfunction called by far_InspirationRoll
--/|   
--/|   far_AnswerNo
--/|     Used to return to calling script if player does not want to use Inspiration or Bardic Inspiration at this time.
--/|   
--/|   far_Bardic|token-id;tokenmarker;Scriptcheck;baseroll;adjustedroll;rollreturn
--/|     Used if PC character has Bardic Inspiration to add at the end of the roll. Also routes to appropriate
--/|     subfunction if using FarsideGallery's Bardic scripts or not.
--/|   
--/|   far_ChooseBard|AttackRoll
--/|     Used to choose the Bard character (token) if not using FarsideGallery's Bardic scripts
--/|   
--/|   far_ScriptBardEntry|token-id;BardicMarker;rollreturn
--/|     Used when using FarsideGallery's Bardic scripts
--/|   
--/|   far_FindBardLevel|Bardtoken;rollreturn
--/|     Subfunction called to determine Bard's level and therefore the dice size added to the referencing roll
--/|   
--/|   far_FIND_CLASS_LEVEL|characterid;class;Return String Var
--/|      Given a (class) name (ie, wizard, warlock, fighter, etc.) will return (Return String Var) with the class level for the 
--/|      character indicated by (characterid). If the character has no levels in that class, the return value will be 0
--/|
--/|   far_DamageTokenmod|tokenid;bar#;tempHP bar#;amount;display (0 or 1);Allow Overhealing (0 or 1); Allow negative hp (0 or 1)
--/|      Applies damage (negative number) or healing (positive number) to (tokenid)'s (bar#) token bar.
--/|
--/|   far_LogFatalError|EmoteText;ErrorHeader;ErrorMessage
--/|
--/|   far_Lib5E_FIND_ACTIVE_GLOBAL_ATTACK_MODS|character_id;StringVariableToSet
--/|   far_Lib5E_FIND_ACTIVE_GLOBAL_DAMAGE_MODS|character_id;StringVariableToSet
--/|   far_Lib5E_FIND_ACTIVE_GLOBAL_SAVE_MODS|character_id;StringVariableToSet
--/|   far_Lib5E_FIND_ACTIVE_GLOBAL_AC_MODS|character_id;StringVariableToSet
--/|   far_Lib5E_FIND_ACTIVE_GLOBAL_SKILL_MODS|character_id;StringVariableToSet
--/|     All of these functions perform similar tasks, so they are grouped here. In each case, supply a Character ID and 
--/|     the name of a string value to be set. The function will read the global mods for the given category and return a string
--/|     representing what to add to a roll expression to account for the mod(s). Dice rolls in mods (ie, 1d4 for guidance) will
--/|     include a # after the expression to keep it from being counted in the Crit/Fumble check.
--/|
